# Flipper-IRDB attribution

The IR database search index and imported remote files used by **SHYS Remote**
come from the community project [Flipper-IRDB](https://github.com/Lucaslhm/Flipper-IRDB)
by Lucaslhm and contributors.

are not covered by the project's current CC0 policy.

## What this integration uses

- **Bundled search index** (`data/irdb_index.json`): curated file paths and metadata
  derived from Flipper-IRDB.
- **Remote import**: individual `.ir` files are downloaded from the Flipper-IRDB
  repository when you select a device.

## Disclaimer

IR signals are community contributions. Device brand names are used only to help
you find compatible remotes. This integration does not claim affiliation with
device manufacturers.
